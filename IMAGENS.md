## 🎯 Dicas de Otimização

### Tamanhos Recomendados
- Imagens grandes: 1200x800px (máximo)
- Imagens médias: 600x400px
- Imagens pequenas: 300x300px
- Avatares: 60x60px

### Formato de Arquivo
- **JPG**: Para fotos e imagens complexas
- **PNG**: Para imagens com fundo transparente e logotipos
- **WebP**: Para melhor compressão (suporte moderno)

### Compressão
Use ferramentas como:
- [TinyPNG](https://tinypng.com/) - Comprime PNG e JPG
- [ImageOptim](https://imageoptim.com/) - Otimização local
- [Cloudinary](https://cloudinary.com/) - Serviço online

### Nomes de Arquivo
- Use nomes descritivos em minúsculas
- Use hífens para separar palavras
- Exemplo: `massagem-nuru.jpg` ✅ / `MassagemNuru.jpg` ❌

## 📱 Resolução para Diferentes Telas

### Desktop (1200px+)
- Imagens podem ser maiores
- Qualidade alta: 100%

### Tablet (768px - 1199px)
- Imagens redimensionadas
- Qualidade: 80-90%

### Mobile (320px - 767px)
- Imagens comprimidas
- Qualidade: 60-75%
